Installation instructions¶

Install angular-froala-wysiwyg from npm

`bash npm install angular-froala-wysiwyg `

You will need CSS styles

`html <!-- index.html --> <link href="node_modules/froala-editor/css/froala_editor.pkgd.min.css" rel="stylesheet"> `
